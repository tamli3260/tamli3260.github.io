Naming Convention

  1. folders & file names with lower case letters
  2. if file name has more than one word, the second word and etc. will start with Upper case letter OR - between the word, e.g: homePage.html, home-page.html
  3. We ususally seperate the files into the following folders"
    • html - for html files
    • images - for images files
    • css - for css files
    • js - for js files
